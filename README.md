# Zapret Lists - Списки для Zapret

Репозиторий содержит списки доменов и IP-адресов для инструмента **Zapret** - программы для обхода интернет-цензуры и блокировок Роскомнадзора.

## Структура файлов

### Основные файлы со списками

| Файл | Назначение | Формат |
| ------ | ----------- | -------- |
| **list-general.txt** | Основной список доменов для обхода блокировок | Доменные имена |
| **list-google.txt** | Специализированный список Google/YouTube доменов | Доменные имена |
| **ipset-all.txt** | Полный список IP-адресов и CIDR блоков, находящихся под блокировкой | CIDR нотация |
| **list-exclude.txt** | Список исключений | Доменные имена |
| **ipset-exclude.txt** | Исключения для IP адресов (локальные сети, зарезервированные адреса) | CIDR нотация |

## Содержание файлов

### list-general.txt

Содержит основные домены сервисов, доступ к которым ограничен:

- Discord (все варианты доменов)
- YouTube и Google сервисы
- Другие популярные сервисы

### list-google.txt

Специализированный список для Google/YouTube сервисов с CDN и API доменами:

- youtube.com, youtu.be
- google.com и подсервисы
- googleapis.com и связанные сервисы

### ipset-all.txt

Полный список IP адресов и подсетей, которые заблокированы.
Основан на реестре запрещённых в России сайтов от ФСТЕК.

### list-exclude.txt

Список российских и важных сервисов, которые должны работать без обхода:

- Яндекс (yandex.com, yandex.ru)
- VK (vk.com, mail.ru)
- Сбербанк, Альфабанк и другие банки
- Российские СМИ и сервисы

## Установка

1. Скачайте `.zip` архив. [>> Скачать <<](https://github.com/Keva1z)

2. Переместите все `.txt` файлы в папку `lists` в zapret и перезапустите сервис.

## Примечание

Списки могут обновляться. Скорее всего не часто. Последние несколько месяцев сижу с ними и всё прекрасно работает.

Использую стратегию `general (ALT 11)`
